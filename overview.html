LTFrame不仅仅是一个界面库,相对于传统的开发方式，它能将程序员、前端开发人员、交互设计师合理的融入在一起协同工作，并且相互之间不需要过多的耦合，便可以开发出高效的应用程序

LTFrame是基于优秀的开源浏览器webkit内核开发而成，使用C/C++语言进行编写，并且对webkti内核做了大量的优化和修改,大幅度提升了webkit的稳定与易用性

What programming language should I use

.net released是基于LTFrame封装而成的.net开放方法，你可以使用C#去写应用程序,但不限于C#,只要可以支持.net framework便可以使用.运行你应用程序的机器只要支持.net 2.0 以上版本便可以运行LTFrame

如果你还不太了解.NET或者C#去这个网站 URL 了解一下

ltframe是使用c++开发而成的，所以用C++开发基于ltframe的应用程序是最好的，并且C++相对与其他语言来讲，开发出的应用稳定性和速度都优于其他语言，如果你可以使用C++可以尝试使用C++来开发LTFrame应用

如果你还不太了解C++或者C去这个网站 URL Learn
