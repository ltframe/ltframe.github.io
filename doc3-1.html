1.virtual LRESULT HandleUserMessage(HWND, UINT, WPARAM, LPARAM)=0;

必须要被重写的函数，可以实现windows消息中自己方法.但在最后,必须要返回WebView中的HandleMessage

2.virtual LPCWSTR GetWindowClassName()=0;

必须要被重写的函数，可以返回当前窗口的类名

3.virtual bool OnDrop(IDataObject* pDataObject, DWORD grfKeyState, POINTL pt, DWORD* pdwEffect){return true;};

当拖动放文件放开鼠标后触发。当你的窗口需要产生拖动文件消息处理时可以重写此函数

4.virtual bool OnDragLeave(){return true;};

当拖动放文件离开窗口后触发。当你的窗口需要产生拖动文件消息处理时可以重写此函数

5.virtual bool OnDragOver(DWORD grfKeyState, POINTL pt, DWORD* pdwEffect){return true;};

当拖动放文件在窗口后触发。当你的窗口需要产生拖动文件消息处理时可以重写此函数

6.virtual bool OnDragEnter(IDataObject* pDataObject, DWORD grfKeyState, POINTL pt, DWORD* pdwEffect){return true;};

当拖动放文件进入窗口后触发。当你的窗口需要产生拖动文件消息处理时可以重写此函数

7.virtual void OnFinalMessage(){};

当你的程序有多个窗口，需要关闭其中一个子窗口时，可以重写次函数，表示在窗口销毁后做一个处理工作,如果要退出应用程序，则不需重写次方法
